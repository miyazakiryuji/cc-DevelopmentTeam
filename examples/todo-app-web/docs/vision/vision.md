# Sharedo (仮称) — プロダクトビジョン

## ひとことで
家族・友人と「やること」をリアルタイムに共有できる Todo アプリ。

## 動機
- 家族で買い物リストを共有したいが、LINE だと流れて埋もれる
- 既存の Todo アプリ (Todoist, TickTick 等) はソロ向け or 高機能すぎる
- 「共有」と「シンプルさ」を両立した選択肢が見当たらない

## ターゲットユーザー
- 主: 同居家族 (夫婦・子育て世帯)
- 副: 同居していないが日常的にやりとりがある友人グループ (旅行計画など)
- 想定規模: 1 リスト = 2〜5 人で共有、1 ユーザー = 1〜3 リスト所属

## 似てるサービス
- **Todoist Shared Projects** / **Microsoft To-Do**: 高機能だが学習コスト高い
- **Google Keep**: メモ風味で軽い、ただしリアルタイム共有が弱い
- **LINE のノート / 投票**: 流れる、Todo 管理に最適化されていない

差別化:
- **リアルタイム反映** (相手が完了マークしたら即見える)
- **ログイン即使える** (タスクリスト作成までの導線が最短)
- **通知は控えめ** (家族の Todo で通知を爆撃したくない)

## これだけは入れたい機能 (MVP)
1. ログイン / 認証
2. リスト作成・編集・削除
3. メンバー招待 (URL or 連絡先)
4. タスク追加・完了マーク・削除
5. リアルタイム同期

## これは絶対やらない
- 通知を爆撃する (リマインダーは最低限)
- 広告表示
- 過剰な分析機能 (生産性スコア等)
- AI レコメンド (まだ要らない)

## プラットフォーム
- フェーズ 1: Web (PC + スマホブラウザ)
- フェーズ 2: モバイルアプリ (PWA からスタート、ネイティブ化は後)

## 期限・規模感
- MVP リリース: 着手から 3 ヶ月以内
- 開発者: 1 人 (副業時間で週末ベース)
