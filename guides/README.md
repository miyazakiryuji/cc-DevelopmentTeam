# cc-DevelopmentTeam 手順書フォルダ

このフォルダには、プラグインを使いこなすための **日本語ガイド** をテーマ別にまとめてあります。
プロジェクトトップの `README.md` から「こんな時はこのガイドへ」と誘導されます。

## ガイド一覧

| ガイド | こんな時に読む |
| --- | --- |
| [quickstart.md](./quickstart.md) | はじめてプラグインを入れて触り始める時 |
| [plans.md](./plans.md) | Claude のプラン（Pro / Max など）を決める時 |
| [mobile-ide.md](./mobile-ide.md) | モバイルアプリ開発で IDE を選ぶ時 |
| [update.md](./update.md) | プラグイン本体を最新版に更新する時 |
| [commands.md](./commands.md) | 各コマンドの使い分けを詳しく知りたい時 |
| [departments.md](./departments.md) | 5 つの部署（agent）の役割を詳しく知りたい時 |
| [workflow.md](./workflow.md) | 業務フローを図解で理解したい時 |
| [directory-structure.md](./directory-structure.md) | プロジェクトのフォルダ構成を確認したい時 |
| [faq.md](./faq.md) | 「これ、どうしたらいい?」をひと通り見たい時 |
| [troubleshooting.md](./troubleshooting.md) | 動かない・思った通りに動かない時 |
| [development.md](./development.md) | このプラグイン自体を改造したい開発者向け |

## 全ガイドの目的

「最初の一歩」「日常運用」「困った時」「拡張したい時」のどれでも、辞書のように引いて使えるよう、テーマごとに切り出しています。

何から読めばいいか迷ったら、まず [quickstart.md](./quickstart.md) → [commands.md](./commands.md) の順番が安心です。
